--------PREGUNTAS--------- 
1. ¿Qué comando se utiliza para clonar un repositorio existente?
Respuesta:
git clone <URL_del_repositorio>

2. ¿Cuál es el comando que sirve para verificar el estado actual del repositorio local?

Respuesta:
 git status

3. ¿Cómo se agregan archivos al área de preparacion?

Respuesta:
git add <nombre_del_archivo>
git add .

4. ¿Qué comando se usa para confirmar los cambios en Git?

Respuesta:
git commit -m "Mensaje de confirmación"
